# AcuBeat Support

Thank you for using AcuBeat\!

## Report an Issue

If you encounter a bug or have a feature request, please [create a new issue](https://github.com/tanizya/acubeat-support/issues/new).

When reporting a bug, please include:
- Device model (e.g. iPhone 15)
- iOS version
- Description of the issue
- Steps to reproduce

## Contact

For other inquiries, please open an issue above.
